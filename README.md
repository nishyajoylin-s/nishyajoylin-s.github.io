# Nishya Website

A simple, clean static resume website built with HTML and CSS.

## Features

- Responsive design (mobile and desktop friendly)
- Professional layout with skills and projects tables
- Career summary section
- LinkedIn profile integration
- Clean, modern styling

## Structure

```
nishya_website/
├── index.html      # Main resume page
├── styles.css      # Stylesheet
└── README.md       # This file
```

## Customization

To personalize this resume:

1. Open `index.html` and replace all placeholder text in brackets `[...]`:
   - `[Your Name]` - Your full name
   - `[Your Title / Tagline]` - Your professional title
   - Skills and projects details
   - Career summary paragraphs
   - `[your-linkedin-username]` - Your LinkedIn profile URL

2. Modify `styles.css` to change colors, fonts, or layout as desired.

## Local Preview

Simply open `index.html` in your web browser to preview the site.
