# Nishya Website

A simple, clean static resume website built with HTML and CSS.

## Features

- Responsive design (mobile and desktop friendly)
- Professional layout with skills and experience sections
- Career summary section
- LinkedIn profile integration
- Clean, modern styling

## Structure

```
nishyajoylin-s.github.io/
├── index.html       # Home page
├── experience.html  # Work experience
├── skills.html      # Leadership, business, and technical skills
├── education.html   # Education background
├── projects.html    # Personal projects
├── blog.html        # Blog
├── portfolio.css    # Stylesheet
└── README.md        # This file
```

## Local Preview

Simply open `index.html` in your web browser to preview the site.

## Deployment

This site is hosted on GitHub Pages at: `https://nishyajoylin-s.github.io/`
