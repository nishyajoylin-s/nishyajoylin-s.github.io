Germany Live Power Grid & Carbon Dashboard
↗ Live app GitHub →

A single-screen, real-time view of the German power system: live generation mix, a computed grid carbon intensity, day-ahead prices, and live green-power for all 16 federal states. Built from public open-data feeds with no API keys and no database, as a power-operations command centre in Streamlit and Plotly.

What it shows, and why it is useful

At a glance: how much of Germany’s electricity is renewable right now, how clean it is, what it costs, and how the 16 states compare. The practical use is timing. A short forecast shows when power will be greenest and cheapest in a given state, so you can pick the best window to charge an electric car or run heavy appliances. For anyone following the energy transition, it is a live, honest read on how the grid is tracking toward the 2030 renewable target, and a clear example of turning open data into something people can act on.

How it is built

  1. Live, key-free feeds. Energy-Charts for the national mix and day-ahead price, Corrently GrünstromIndex for per-postcode green power, Open-Meteo for solar and wind.
  2. Carbon computed, not fetched. No CO₂ endpoint exists, so grid carbon intensity is derived from the live mix using IPCC lifecycle emission factors.
  3. All 16 federal states. The green-power feed is keyed by postcode, so each state is sampled at its capital for even national coverage.
  4. Live solar field. One batched Open-Meteo call returns a ~300-point irradiance grid, rendered as a heat layer behind the state bubbles.
  5. Command-centre layout. A single-screen, Z-pattern design: headline KPIs across the top, the map as the anchor, the actionable “when is power greenest” forecast bottom-right.
  6. Simple to run. No database to manage; pushed to GitHub and served on Streamlit Community Cloud from a single file.
